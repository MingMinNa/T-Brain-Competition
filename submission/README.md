### 其餘資料
* **```upload_template.csv```**  
    競賽方提供的上傳檔案格式。
* **```q_date.csv```**  
    基於```upload_template.csv```，將相同日期和地點的資料做合併，並使用**日期、地點**的格式儲存。
* **```output.csv```**  
    模型讀取 ```q_datte.csv``` 資料後，生成的答案。  
    需要再使用```related_data.py```中的```build_answer```函數，將其轉成 ```answer.csv```。
